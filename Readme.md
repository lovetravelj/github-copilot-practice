
# GitHub Copilot 과정 평가 안내


## 평가 배점

- 테스트 : 객관식 20문항 (70점)
- Vibe 코딩 프로젝트 (30점, Pass/Fail)
- 뛰어난 프로젝트의 경우 5점 가점 (최대 5개 프로젝트 선정)



## Vibe Coding 프로젝트 평가 안내

### 1. 프로젝트 평가 개요

GitHub Copilot을 활용하여 **간단하지만 실제로 사용할 수 있는 유용한 앱**을 기획·개발합니다.

---

## 2. 필수 구현 사항

다음 요소가 반드시 포함되어야 합니다:

*  프로젝트의 **목표와 필요성**
*  **핵심 로직**
*  핵심 로직에 대한 **테스트 코드**
*  **API** (요청/응답 구조 포함)
*  **UI**
*  실제로 **실행 가능한 상태**

## 3. 제출물

* GitHub Repository
* README
  * 프로젝트 설명
  * 실행 방법
  * 핵심 로직 / API 설명
* 테스트 코드 포함
* 실행 화면 캡처

## 4. 평가 기준

* **Pass / Fail (30점 / 0점)**
* 뛰어난 프로젝트의 경우 **5점 가점** (최대 5개 프로젝트 선정)
